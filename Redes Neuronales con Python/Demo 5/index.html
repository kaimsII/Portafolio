Demo 5: Extracción de Conocimiento

Texto • Grafos • Probabilidad • Tiempo

5.1 Minería de Texto

Paso 1: Entender el Lenguaje.
El mundo está lleno de textos (correos, tuits, documentos). Para que una IA los entienda, primero debemos limpiarlos y convertirlos en "fichas" contables.

El Reto: El texto es caótico y ruidoso.

La Solución: "Limpiamos" lo inútil y nos quedamos con las palabras clave.

Ya tenemos las palabras clave, pero...
¿Qué pasa cuando los datos se conectan entre sí?

5.2 Análisis de Grafos

Paso 2: Mapear Relaciones.
Las palabras, personas o páginas web no están solas. Forman redes. Aquí analizamos quién es "influencer" (nodo central) y cómo fluye la información.

El Reto: Entender la estructura oculta de la red.

La Solución: Medir quién tiene más conexiones (Centralidad).

Entendemos la estructura, pero...
¿Cómo tomamos decisiones cuando hay incertidumbre?

5.3 Probabilidad (Naive Bayes)

Paso 3: Decidiendo con Pistas.
Imagina que eres un detective. No sabes si un correo es Spam o Seguro, así que buscas pistas (palabras).

Cada palabra es una "evidencia" que empuja la balanza hacia un lado u otro.

Ejemplo: Correo sospechoso.

Pista: "Ganar" → +Spam

Pista: "Proyecto" → +Seguro

Resultado: Combina todas las pistas para decidir.

Podemos clasificar el presente, pero...
¿Podemos predecir el futuro?

5.4 Series Temporales

Paso 4: Ver el Futuro.
Los datos reales (ventas, clima) son ruidosos y caóticos. Para predecir el futuro, no podemos seguir el caos.

Debemos descubrir el patrón oculto (la tendencia limpia) y proyectarlo hacia adelante.

El Reto: El "Ruido" esconde la verdad.

La Solución: Ajustar una curva suave al pasado y extenderla.
