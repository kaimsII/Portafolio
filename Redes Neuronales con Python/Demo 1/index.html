Guía Rápida de Activación

ReLU Ideal para Círculos. Rápido y eficiente, pero falla en patrones complejos como XOR.
Tanh Ideal para XOR. Captura curvas complejas gracias a su rango (-1 a 1).
Sigmoid Clásica (0 a 1). Útil para probabilidades suaves, pero converge más lento.
